# Fillflow AI Website

A single-file HTML website for Fillflow AI — an AI lead management service for Australian dental practices.

## File

- `Fillflow_AI_Website.html` — the entire site (HTML, CSS, and JavaScript in one file)

## Pages

- **Home** — hero, problem section, how it works, results, who it's for
- **Services** — detailed breakdown of each service offering
- **About** — origin story, values, and why dental practices
- **Contact** — enquiry form and contact details

## Running Locally

Open `Fillflow_AI_Website.html` directly in any modern browser. No build step or server required.
